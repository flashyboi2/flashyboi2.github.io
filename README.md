# flashyboi2.github.io

Hello, this is my first deployment of this simple calendar based website, more like a barebones version of ToDoIst

You can add tasks to calendar, edit and track them, the essential features. The data is stored locally and doesn't require any
subscriptions for this. So you can add as much as you can. Works offline too.

UI design was entirely done by AI and the javascript program was done by me.
